## 搭建 Vue 3 + Capacitor Android 开发环境

### 完成内容
- 在 `C:/Users/28166/WorkBuddy/安卓/` 搭建了完整的 Vue 3 + Capacitor Android 开发环境
- 技术栈：Vue 3 + Vite 6 + TypeScript + Vue Router 4 + Pinia 3 + Vant 4 + Capacitor 7

### 项目结构
- `src/` - Vue 源码（main.ts, App.vue, router/, stores/, views/）
- `android/` - Capacitor 生成的 Android 原生工程
- `capacitor.config.ts` - Capacitor 配置（appId: com.example.androidvueapp）
- `scripts/` - 开发辅助脚本

### 关键问题：中文路径 npm 兼容
- 工作区路径含中文（"安卓"），npm/node 无法正常运行
- 解决方案：创建 `scripts/` 辅助脚本，将源码同步到 `/tmp/android-vue-temp`，在临时目录执行所有 npm 操作，完成后同步产物回工作区
- 常用命令：
  - `bash scripts/dev.sh` - 启动开发服务器
  - `bash scripts/build.sh` - 构建 Web 应用
  - `bash scripts/install.sh` - 安装/更新依赖
  - `bash scripts/build-android.sh` - 构建 Android（需 Android Studio）
